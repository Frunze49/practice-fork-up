## [01-git] Hometask
Команда get на Java. Использовала javac version 11.0.20.

Build:
```bash
javac GitGet.java
```

Usage:
```bash
java GitGet <path> <root_hash>
```

#### Example:

command:
```bash
java GitGet subdir/text.txt 99682225f8eeb6a007bed389c124c659f596a794d556b4c865c99995c5f03b98
```

output:
```bash
Just a text file
```
